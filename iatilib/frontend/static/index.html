About

Human-readable information about the API.

/api/1/about

Activity: Basic Query

List activity objects. Note fields such as total (number of results), last_page, per_page etc.

/api/1/access/activities
/api/1/access/activities?per_page=50
/api/1/access/activities?per_page=20&page=7

Activity: Query by Country

Search <recipient-country> text, or look up by ISO code.

/api/1/access/activities?country_code=LY
/api/1/access/activities?country=libya

Activity: Query by Reporting Organisation

Search <reporting-org> text, or look up by ref attribute.

/api/1/access/activities?reporting_org_ref=GB-CHC-285908
/api/1/access/activities?reporting_org=world%20bank

Activity: Query by Participating Organisation

Search <participating-org> text, or look up by ref attribute.

/api/1/access/activities?participating_org_ref=44002
/api/1/access/activities?participating_org=peace

Activity: Query by Sector

Search for activities in a given sector.

/api/1/access/activities?sector=sanitation
/api/1/access/activities?sector_code=12230

Activity: Query by Date

Search for activities occurring on a given date. The date must fall between start-actual (or start-planned) and end-actual (or end-planned).

/api/1/access/activities?date=2009-05-09
/api/1/access/activities?date=1995-01-01Z15:30:00

The IATI API is updated nightly. This is a collaboration between IATI and the Open Knowledge Foundation.
